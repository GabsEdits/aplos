Aplós

Aplós is a simple, modern, and practical by design theme build up on Aeolus. The idea is a simple way to have a quick page on the web. It aims to provide all the needed options for comfortable writing, keeping the balance of it being simple.

• Simple and lightweight. No need for JavaScript.
• For a very pleasant look, the colors are tinted with the primary color.
• Proper favicon for modern browsers and Apple device icons.
• Social media meta cards for easy sharing.
• One of the best fonts: Inter and Source Code Pro. Everything local using NPM.
• Mobile friendly, with dark mode.

“Simplicity is the ultimate sophistication.”
— Leonardo da Vinci

Install.

Clone it locally

git clone https://github.com/

Open it, and change everything that is in the docs there. And Done!
